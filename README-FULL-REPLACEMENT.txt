DAVID TRAVEL FULL REPLACEMENT PACKAGE

This package replaces the active app/site code with one consistent version.

It includes:
- Homepage
- Blog page
- Single post pages
- Categories
- Search
- Legal/static pages
- Header
- Footer
- Sidebar
- Article cards
- Sanity Studio route
- Sanity config with Structure tool restored
- Local MDX posts fallback
- Sanity posts loaded first
- Logo files

INSTALL:
1. Unzip this package.
2. Copy ALL folders/files into:
   Documents/GitHub/david-travel-next
3. Choose Replace/Merge.
4. Run:
   cd ~/Documents/GitHub/david-travel-next
   git add .
   git commit -m "Full replacement stable site"
   git push
5. In Hostinger, redeploy/pull latest from GitHub.

IMPORTANT:
Do not delete the existing content/posts folder. This package does not include all old posts.
It reads your existing content/posts folder plus Sanity posts.

POSTING:
Use:
https://daviddottravel.com/studio
