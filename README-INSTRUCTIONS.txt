Copy these files/folders into your david-travel-next project folder and choose Replace when prompted:

- sanity.config.ts
- tsconfig.json
- next.config.mjs
- app/studio/[[...tool]]/page.tsx

After copying, open Terminal and run:

cd ~/Documents/GitHub/david-travel-next
rm -rf .next
npm run dev

Then open:
http://localhost:3000/studio
